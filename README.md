# YouTube Clone

A responsive front-end clone of the YouTube desktop interface, built using pure HTML and CSS. This project focuses on mastering modern CSS layout techniques, including Flexbox and CSS Grid, to create a structured and scalable design.

## Features
* **Responsive Navigation:** A fixed header with a functional search layout and consistent icon set.
* **Modular CSS Architecture:** Styles are organized into specific modules (`general.css`, `header.css`, `sidebar.css`, `video.css`) for better maintainability and code readability.
* **Grid-based Layout:** Utilizes CSS Grid for the video feed to ensure a clean, modern look.
* **Component-based Structure:** Separate handling for sidebars, thumbnails, and channel data.

## Technologies Used
* **HTML5:** Semantic structure for content.
* **CSS3:** Flexbox, CSS Grid, and custom styling for responsive components.
